MASA API'sini projenize entegre etmek için aşağıdaki adımları izleyin. API anahtarınızı API Anahtarları bölümünden alabilirsiniz.
npm / yarn
npm install @masa/api-client # ya da yarn add @masa/api-client
JavaScript / Node.js
JavaScript
import MasaClient from '@masa/api-client'; const masa = new MasaClient({ apiKey: 'masa_live_sk_k8Rp2xLmQnV4wZ9cYuDjHbT3' }); // Restoranları listele const restaurants = await masa.restaurants.list({ neighborhood: 'Beşiktaş', cuisine: 'Steakhouse' }); // Rezervasyon oluştur const reservation = await masa.reservations.create({ restaurantId: 1, date: '2026-03-25', time: '20:00', people: 2, tableId: 5 }); console.log('Rezervasyon:', reservation.id);

📄 API Dokümantasyonu
MASA REST API — v1.0
Base URL: https://api.masa.com.tr/v1
Restoranlar
GET/api/restaurants
Tüm restoranları listeler. Opsiyonel: ?neighborhood=Beşiktaş&cuisine=Balık&price=2
GET/api/restaurants/:id
Tek restoran detayını döner. Menü, saatler, masa durumu dahil.
GET/api/restaurants/:id/availability
Belirli tarih ve saat için müsait masaları döner.
Rezervasyonlar
POST/api/reservations
Yeni rezervasyon oluşturur. Body: { restaurantId, date, time, people, tableId, preorderItems }
GET/api/reservations/:id
Rezervasyon detayını döner.
DELETE/api/reservations/:id
Rezervasyonu iptal eder. 24 saat öncesine kadar ücretsiz iptal.
Kimlik Doğrulama
Authorization: Bearer masa_live_sk_...
Her istekte Authorization header gönderilmelidir.
